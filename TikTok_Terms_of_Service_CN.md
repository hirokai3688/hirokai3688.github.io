## 2. 服务描述
本应用是一个 TikTok 第三方AI助手工具，提供以下功能：
- TikTok 内容分析和数据洞察
- 粉丝互动管理和分析
- 内容发布计划和优化建议
- 通过 TikTok Open API 安全访问您的 TikTok 账户数据

## 3. TikTok API 使用
本应用使用 TikTok Open API 访问您的账户数据。使用本应用即表示您授权我们通过 TikTok API 访问：
- 您的基本个人资料信息（用户名、头像、简介）

- 您的粉丝和关注列表统计信息
- 视频互动数据（点赞、评论、分享数量）
- 账户分析数据（仅用于提供分析服务）
