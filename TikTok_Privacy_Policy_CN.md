# 隐私政策

## 1. 引言
OpenClaw Linky TikTok 助手（以下简称"本应用"）重视用户隐私。本隐私政策说明我们如何通过 TikTok API 收集、使用和保护您的信息。

## 2. 信息收集范围
当您授权本应用访问 TikTok 账户时，我们通过 TikTok Open API 收集以下信息：

### 2.1 基本账户信息
- TikTok 用户 ID 和用户名
- 个人资料信息（头像、显示名称、简介）
- 账户验证状态和类型

### 2.2 内容信息
- 发布的视频列表和元数据
- 视频标题、描述、标签
- 视频统计信息（观看数、点赞数、评论数、分享数）

### 2.3 社交信息
- 粉丝数量统计
- 关注者数量统计
- 粉丝互动趋势数据

### 2.4 使用数据
- 应用功能使用频率
- 功能偏好设置
- 服务请求日志（匿名化）

## 3. 信息使用目的
我们使用收集的信息仅用于：

### 3.1 核心服务提供
- 生成内容分析报告
- 提供粉丝互动洞察
- 优化内容发布建议
- 个性化AI推荐

### 3.2 服务改进
- 分析功能使用情况以优化用户体验
- 识别和修复技术问题
- 开发新功能

### 3.3 安全和合规
- 检测和防止滥用行为
- 确保服务安全稳定
- 遵守法律和平台要求

## 4. 数据存储和安全

### 4.1 存储方式
- 数据加密存储在安全服务器
- 敏感信息额外加密保护
- 定期备份和灾难恢复

### 4.2 保留期限
- 服务数据保留至用户删除账户或撤销授权
- 匿名化分析数据可能长期保留
- 日志数据定期清理（最长90天）

### 4.3 安全措施
- HTTPS 加密传输
- 服务器访问控制
- 定期安全审计
- 员工隐私培训

## 5. 数据共享和披露

### 5.1 我们不会：
- 出售您的个人数据
- 与广告商共享识别信息
- 将数据用于本政策未说明的目的

### 5.2 仅在以下情况共享：
- **法律要求**：响应法院命令或法律程序
- **保护权利**：防止欺诈、滥用或安全威胁
- **服务提供商**：与必要的云服务商共享（受保密协议约束）

## 6. 用户权利和控制

### 6.1 您的权利
- **访问权**：查看我们持有的您的数据
- **删除权**：请求删除您的个人数据
- **更正权**：更正不准确的信息
- **撤回权**：随时撤销 TikTok API 访问权限

### 6.2 如何行使权利
1. 通过应用内设置管理数据
2. 通过 TikTok 设置撤销应用权限
3. 发送邮件至 hirokai@example.com 提出请求
（请替换为实际联系邮箱）

## 7. TikTok 平台集成
- 本应用遵守 TikTok 开发者政策和API使用条款
- 您通过 TikTok OAuth 流程授权数据访问
- 授权范围明确显示在授权页面
- 您可以随时在 TikTok 设置中管理第三方应用权限

## 8. 儿童隐私
本应用不面向13岁以下儿童。如果我们发现收集了儿童数据，将立即删除。

## 9. 国际数据传输
数据主要存储在所在地区服务器，可能涉及跨境传输。我们确保传输符合适用数据保护法律。

## 10. 政策更新
我们可能更新本政策以反映服务变化或法律要求。重大变更将：
- 通过应用内通知告知
- 更新页面顶部的"最后更新日期"
- 给用户合理时间审阅变更
